## Branches
- `production`
    - 실 서비스에 올라간 코드
- `master`
    - 실 서비스에 올라가도 문제가 없는 코드
- `dev` `feature/~`
    - 개발 중인 코드

## 시작하기
### 의존성 설치
`npm install` (으)로 프로젝트 빌드에 필요한 의존성 패키지를 다운로드합니다.

### 개발 모드 시작
`npm run start` (으)로 개발 모드로 프로젝트를 Build & Run 합니다.

### 빌드
`npm run build` (으)로 production build를 합니다.

### ESLint 검사
`npm run lint` (으)로 ESLint 검사를 진행합니다.
